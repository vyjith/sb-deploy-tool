# This is for just demo for sb-deploy command

```
git clone https://github.com/vyjith/sb-deploy-tool.git
```
```
cd sb-deploy-tool
```
```
chmod +x j.sh
```
```
echo "alias sb-deploy='bash $(pwd)/j.sh'" >> ~/.bashrc
```
```
source ~/.bashrc
```
